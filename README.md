# Student Performance Analytics Dashboard

## Project Overview
This project analyzes student academic performance using data analytics and visualization techniques. 
An interactive dashboard was developed to help educators understand student engagement and performance patterns.

## Problem Statement
Educational institutions often have large volumes of academic data but lack effective visualization tools to analyze student performance. 
This project aims to transform raw student data into meaningful insights through interactive visual analytics.

## Objectives
- Analyze student academic performance
- Identify patterns in attendance, study hours, and LMS usage
- Visualize correlations between engagement metrics and final grades
- Support data-driven academic decision making

## Dataset
The dataset contains student engagement and performance attributes such as:

- Attendance Percentage
- Study Hours per Week
- LMS Hours
- Final Grade
- Gender

## Data Processing
The following preprocessing steps were performed:

- Removal of duplicate records
- Handling missing values
- Outlier detection using IQR method
- Data normalization using MinMaxScaler
- Feature engineering (Engagement Score)

## Methodology
The analysis includes:

1. Descriptive Analytics
2. Diagnostic Analytics
3. Correlation Analysis
4. Predictive Modeling (Linear Regression)

## Dashboard Features
- Student engagement visualization
- Correlation heatmap
- Performance comparison charts
- Interactive data filtering
